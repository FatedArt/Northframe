# Northframe

Modular UI/UX productivity toolkit: design system audits, persona workflows (planned), and more.

## Modules

| Module | Description |
|--------|-------------|
| [DesignSystemAudit](./DesignSystemAudit/) | Local Flask dashboard to audit Figma design systems (scope: `Folder:` frames), export markdown reports. |

## Requirements

- Python 3.10+ (per module)
- [Figma Personal Access Token](https://www.figma.com/developers/api#access-tokens) where applicable

## Contributing

Add new tools as top-level folders under this repository with their own `README.md` and `requirements.txt` when needed.
