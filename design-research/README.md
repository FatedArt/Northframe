# Design research

Workflow berbasis file Markdown: isi **INSTRUCTIONS.md** sebagai panduan, kerjakan output di **SANDBOX.md** (atau salin struktur ke dokumen tim).

| Folder | Fungsi |
|--------|--------|
| [PersonaCreation](./PersonaCreation/) | Menyusun persona dari temuan riset. |
| [ProblemStatement](./ProblemStatement/) | Merumuskan problem statement yang jelas. |
| [ResearchSummary](./ResearchSummary/) | Merangkum hasil riset untuk handoff ke desain/produk. |

Semua modul ini bersifat *prompt/template*; tidak memerlukan runtime tambahan.
