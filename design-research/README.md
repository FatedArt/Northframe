# Design research

Workflow berbasis file Markdown: isi **instructions.md** sebagai panduan, kerjakan output di **sandbox.md** (atau salin struktur ke dokumen tim).

| Folder | Fungsi |
|--------|--------|
| [persona-creation](./persona-creation/) | Menyusun persona dari temuan riset. |
| [problem-statement](./problem-statement/) | Merumuskan problem statement yang jelas. |
| [research-summary](./research-summary/) | Merangkum hasil riset untuk handoff ke desain/produk. |
| [prd-ux-heuristic-evaluation](./prd-ux-heuristic-evaluation/) | Product requirements document siap eksekusi untuk UX Heuristic Evaluation App (MVP web-only). |

Semua modul ini bersifat *prompt/template*; tidak memerlukan runtime tambahan.
